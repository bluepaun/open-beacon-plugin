## What I do
- Use `beacon_search` before broad grep-style searches.
- Prefer ranked Beacon matches with file paths and line ranges.
- Fall back to direct file reads only after Beacon narrows the search area.

## When to use me
Use this skill when the task is about finding implementation details by meaning rather than exact string matching.
